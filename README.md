# PlaceMate - Placement Preparation Assistant Chatbot

## Objective
PlaceMate is a chatbot developed using Dialogflow ES to help engineering students prepare for placements.

## Features
- Aptitude Preparation
- Programming Guidance
- Resume Tips
- Interview Preparation
- Placement Eligibility Information

## Technologies Used
- Dialogflow ES
- NLP
- Intent-based Chatbot Design

## Intents
1. Welcome Intent
2. Aptitude Intent
3. Programming Intent
4. Resume Intent
5. Interview Intent
6. Eligibility Intent
7. Exit Intent
